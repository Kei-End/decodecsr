# ECC CSR Assistant & CSR Decoder

This package keeps the original single-page UI and packages it for Cloudflare Pages.

## Included
- Environment, Certificate, Output, Decoder, Post-Deployment Validation, and Knowledge Bank tabs
- Separate output tabs for:
  - Linux OpenSSL
  - Windows PowerShell
  - Windows IIS GUI
- Local browser CSR decoder
- Improved CSR signature verification logic

## Notes
- This build is static and Cloudflare Pages friendly.
- The Decoder tab remains local-browser only.
- No Functions are included.
